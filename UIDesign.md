# UI 设计方案：我的核心资产（定制版）

  

## 设计定位

  

**用户画像：**

- 使用频率：一天至少 3 次（高频）

- 核心关注：仓位情况（第一优先级） > 估值状态（第二优先级）

- 风格偏好：浅色系 + 浅蓝色调

- 布局要求：清晰度 > 信息密度，宁愿少看一点

  

**设计策略：**

- 浅蓝色主题，清新专业

- 仓位信息视觉突出（大字号 + 醒目位置）

- 一屏显示 3-4 个资产卡片（清晰度优先）

- 估值状态用色块标识，一眼可见

- 大字号 + 充足留白

**核心设计要点：**

  

#### 1.1 总资产卡片（高度 ~160pt）

- **白色背景 + 浅蓝色边框**（2pt）

- 总资产数字：52pt，粗体，深灰色

- 整体仓位用**蓝色进度条**突出显示

- 圆角：16pt

  

#### 1.2 资产卡片（高度 ~200pt）★★★

- **白色背景 + 轻微阴影**

- **左侧估值色条**（宽度 12pt，高度 100%）：

- 🟢 绿色 = 极度/比较低估

- ⚪ 灰色 = 合理

- 🟡 橙色 = 比较高估

- 🔴 红色 = 极度高估

  

- **仓位卡片**（核心！）：

- 浅蓝色背景 (#E3F2FD)

- 占据卡片中部约 1/3 空间

- 大字号显示"当前 XX% 目标 XX%"（28pt）

- 蓝色实心进度条（当前仓位）

- 灰色虚线（目标仓位）

- 底部智能提示：

- 超出目标 > 3%：⚠️ 橙色警示

- 超出上限：🚨 红色警示

- 接近目标（±2% 以内）：✅ 绿色

- 低于目标：📊 蓝色提示

  

- **其他信息层级**：

- 第一层：资产名称 + 代码（20pt）

- 第二层：当前价格（32pt，加粗）

- 第三层：仓位卡片（核心）

- 第四层：持仓信息（16pt，灰色）

- 第五层：操作按钮

  

#### 1.3 操作按钮

- 宽度各占 45%，间距 10%

- 高度 44pt

- 圆角 12pt

- [买入]：浅绿背景 + 绿色文字

- [卖出]：浅红背景 + 红色文字

  

#### 1.4 布局策略

- 卡片间距：16pt

- 一屏显示：总资产卡片 + 3 个资产卡片（刚好填满）

- 向下滑动查看更多资产

**设计要点：**

  

#### 2.1 页面结构

从上到下优先级：

1. **价格卡片**（必看，快速了解）

2. **★ 仓位分析**（核心关注，最大篇幅）

3. **估值状态**（决策参考）

4. **持仓详情**（盈亏数据）

5. **最近交易**（历史记录）

  

#### 2.2 仓位分析卡片（重点设计）

- **浅蓝色背景**，突出重要性

- **超大字号显示当前仓位**（48pt）

- **三条进度条可视化**：

1. 蓝色实心条：当前仓位

2. 灰色虚线：目标仓位

3. 红色虚线：仓位上限

  

- **仓位偏离提示框**（仅显示偏离数值，不给操作建议）：

- 超出目标：橙色框 + 偏离百分比

- 超出上限：红色框 + 警示

- 接近目标：绿色框

- 低于目标：蓝色框 + 偏离百分比

  

#### 2.3 估值状态卡片

- 简化设计，不抢仓位的风头

- 大标签显示当前状态

- 简单的区间图即可

```swift
func positionColor(current: Double, target: Double, max: Double) -> Color {

let deviation = current - target

  

if current >= max {

return Color(hex: "#E53935") // 红色：超出上限

} else if deviation >= 5 {

return Color(hex: "#FB8C00") // 橙色：严重超标

} else if deviation >= 3 {

return Color(hex: "#FFA726") // 浅橙：超标

} else if abs(deviation) <= 2 {

return Color(hex: "#43A047") // 绿色：接近目标

} else {

return Color(hex: "#1E88E5") // 蓝色：正常范围

}

}

```

## 总结

  

### 核心设计特点

  

1. **仓位优先**：

- 仓位信息占据最醒目位置

- 浅蓝色卡片突出显示

- 大字号 + 可视化进度条

- 智能建议系统

  

2. **浅蓝主题**：

- 清新专业，符合金融应用调性

- 视觉舒适，适合频繁查看

  

3. **清晰布局**：

- 一屏 3-4 个资产卡片

- 充足留白，降低视觉疲劳

- 信息层级分明

  

4. **高频使用优化**：

- 首页直达核心信息

- 筛选功能快速定位

- 操作流程简洁高效
